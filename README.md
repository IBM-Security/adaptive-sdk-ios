# IBM Security Verify Adaptive SDK for Android and iOS

This repository contains two sample apps; a Kotlin sample app using the IBM Security Verify Adaptive SDK for Android, and a Swift sample app using the IBM Security Verify Adaptive SDK for iOS. The purpose of these sample apps is to showcase the IBM Security Verify Adaptive SDKs for the mobile platforms, and provide guidance when developing mobile applications with it.

## Prerequisites

- Download the IBM Security Verify Adaptive SDK for Android or iOS from the [IBM App Exchange](https://exchange.xforce.ibmcloud.com/hub/IdentityandAccess).

- Download and unzip the Trusteer SDK for Android or iOS after [application onboarding](https://docs.verify.ibm.com/verify/docs/on-boarding-a-native-application).
